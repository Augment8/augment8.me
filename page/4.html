広島Git勉強会 201306

こんにちは。火村です。

広島 Git 勉強会 201306を行いました。私も1セッション担当したり、司会をしたり、運営をしたりしました。

Git というのはアプリケーションやシステムの開発に利用をするツールです。バージョン管理システムと言われる種類に属するものになります。
非常に便利なツールなのですが、一緒に開発する人たちが使えなければ、便利さが半減してしまうため、この Git というツールの普及活動をしています。

今回は広島工業大学さんを会場に学生さんたちを巻き込みつつの勉強会となりました。勉強会のスタイルとしては、発表者が各々勉強してきたことを発表するというスタイルです。具体的にどんなセッションがあったかというと Wiki のほうに記載しています。もし興味があればご覧ください。

私が担当したセッションは 「やりなおせるGit入門」 というタイトルで、”入門した際に体験するチュートリアルの中で、間違えたときはどうすればいよいか、どのようなことが起きるか、また、元に戻すにはどんな危険があるのか。“という点に着目して、Git というツールの動きを紹介しました。

スライドを無料で公開しています。こちらのページが はてなブックマーク というサービスで 400 を越えるブックマークをいただきました。本当にありがとうございます。

これからもいろんな勉強会でセッションしていきたいと思います。
もし見かける機会があればよろしくお願いします。

2013.5.21.火.

すごい広島 という ミートアップをはじめます

こんにちは、火村です。

すごい広島 という ミートアップをはじめます。

このイベント すごい広島 は、ITに関連したエンジニアやデザイナさんが毎週集まっている場所を作ろうというのが目的のひとつです。
毎週水曜日 の 19:00 – 21:50 に 本通りの タリーズコーヒー に集まっています。

いきさつ

広島にも様々なIT系の勉強会は増えてきました。
しかし、認知度はまだまだ高くはないです。
毎週行なうことで、

  • 勉強会の存在を目にする機会を増やしたり
  • 勉強会の存在を知ったとしても月一では都合が合わないという人が、毎週やることで勉強会に参加している人に話を聞くことができたり

するきっかけを増やしたい。という考えです。

このイベント自体は Asakura.rb や Okayama.rb というイベントに触発されています。これらのイベントは毎週行なわれていて、評判も良いです。
どのようなやり方をしているのかはコミュニティそれぞれですが、すごい広島では気楽に参加できることに重点を置いて運営していく予定です。

しかし、目的もなしに人を集めるのは難しいです。
上記のイベントでは Rubyというプログラミング言語を中心に据えています。
そこで「すごい広島」にも何か目的を作ることで、方向性を持たせることにしました。

目的とルール

「すごい広島」では、活動をアウトプットするを目的にすることにしました。
参加者のアウトプットがわかるので、お互いに話題にするネタもみつけることができます。
また、アウトプットをすることで「すごい広島」や「各個人」の知名度を上げていくことができます。
そして、時間で何かを行い、公表するという作業を繰り返すことで、自分の成果物がどんどん残していくことができます。

そのためのルールとして、

  • ブログ等でアウトプットして記事をかく
  • すごい広島 のサイトからリンクを貼る
    • というのをルールにしました。

    これを行なうためのフローとして Github というサービスを利用をすることにしました。これを機会に Github というサービスに興味を持ってもらったり、触る機会も増やしています。

    少し詰め込みすぎてるいる感じはありますが、これらのルールに絶対に従う必要はありません。
    参加者のできる範囲でできることをしてください。

    最後に

    こういったイベントを増やしていくことで、広島の街が盛り上がるきっかけの一つになればと思います。

    もし興味があれば、毎週行っていくつもりなので、遊びに来ていただけたらと思います。

「A8W:1 〜プロジェクションマッピング入門〜」終了しました

プロジェクションマッピング入門のワークショップ、終了しました。
参加いただいたみなさん、ありがとうございました。
そして長時間お疲れ様でした。

オグハチとしては初のスペシャリストを講師に迎えたワークショップでしたが、
実際に運営するにあたり、どのような注意点が必要かといった話から、
手始めに白い箱（立方体）を対象としてやってみるテクニカルな部分まで、
ざっと全体像を見渡せた内容だったのではと思います。

来場者アンケートでもみなさんから好評価をいただき、
中にはもっと突っ込んだ内容も聞きたいという意見もありました。
さっそく応用編として次回開催の時期を検討中です。

今後もこういった各分野で活躍されているスペシャリストを
講師に迎えたワークショップを開催する予定です。
興味のあるテーマや内容のときは、気軽に参加ください。
