CSS Nite in HIROSHIMA Vol.7 を開催しました

こんにちは。火村です。

オグハチのメンバーの多くがスタッフとして参加しているCSS Nite in HIROSHIMA Vol.7が開催されました。
CSS Nite（シーエスエス・ナイト）は、Web制作に関わる方のためのセミナーイベントで広島でも、すでに6度開催されています。

今回のテーマは「プロジェクトを成功にうまく導く秘訣」としまして、県内外の講師のセッションがありました。今回の目玉のセッションである高畑さんの「発注側はこんなロジックで予算を決めている」は、大変好評をいただきました。

本イベントを成功させるためにオグハチのメンバーは副実行委員長や会計、幹事など役割を担当していました。これからも地域のセミナーを応援していきたいと思います。

広島Ruby 勉強会 #033

こんにちは。火村です。

概ね毎月行っていますが、広島Ruby勉強会 #033 を開催しました。
この勉強会は特にテーマは決めておらず、近年はセッションの練習することを主眼を置いています。自分の勉強したことを他の人に伝えることで、自分が勉強するというスタイルです。

Ruby というプログラミング言語を題材にしているので毎回1セッションほど、基礎的な話をするようにしています。私はこれを毎回担当しています。また、これに加えて20分ぐらいのセッションを行いました。「Rails プロジェクトでスタートダッシュを決める」というタイトルでスライドはこちらにあります。

大きなセミナーで講師をするのも大事なのですが、挑戦的なことや繰り返し行うことで、わかりやすい説明ができるように努力していきたいと思います。小さなことからコツコツと。

次回は 9月7日 を予定しています。
参加費は無料です。興味があれば、ぜひ。

2013.6.02.日.

広島Git勉強会 201306

こんにちは。火村です。

広島 Git 勉強会 201306を行いました。私も1セッション担当したり、司会をしたり、運営をしたりしました。

Git というのはアプリケーションやシステムの開発に利用をするツールです。バージョン管理システムと言われる種類に属するものになります。
非常に便利なツールなのですが、一緒に開発する人たちが使えなければ、便利さが半減してしまうため、この Git というツールの普及活動をしています。

今回は広島工業大学さんを会場に学生さんたちを巻き込みつつの勉強会となりました。勉強会のスタイルとしては、発表者が各々勉強してきたことを発表するというスタイルです。具体的にどんなセッションがあったかというと Wiki のほうに記載しています。もし興味があればご覧ください。

私が担当したセッションは 「やりなおせるGit入門」 というタイトルで、”入門した際に体験するチュートリアルの中で、間違えたときはどうすればいよいか、どのようなことが起きるか、また、元に戻すにはどんな危険があるのか。“という点に着目して、Git というツールの動きを紹介しました。

スライドを無料で公開しています。こちらのページが はてなブックマーク というサービスで 400 を越えるブックマークをいただきました。本当にありがとうございます。

これからもいろんな勉強会でセッションしていきたいと思います。
もし見かける機会があればよろしくお願いします。
