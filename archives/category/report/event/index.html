2014.2.03.月.

「オープンセミナー2014@広島」を開催しました

こんにちは。火村です。

2014年2月1日にオープンセミナー2014@広島を開催しました。

オープンセミナーはIT技術者向けの無料のセミナーで、私は実行委員長という立場で、テーマ決め、講師への講演依頼、司会、ウェブサイト更新、受付補佐、会計などを担当しました。講師やスタッフの助けもあり、目標に設定した参加者90名を越えて93名の参加者がありました。

オープンセミナー2014@広島は、テーマを「TDD…そして Be Agile」という「テスト駆動開発」と「アジャイルになる」というふたつのテーマを設定しました。あとは講師のセッション内容を参考に、順番を決めたのですが、講師のみなさんがうまく連携してテーマ性のあるイベントになりました。

TDDはまだまだ開発者にはまだまだ浸透しておらず、利用も難しいと思われている開発手法ですが、仕事や成果物の健康を保つ方法として注目されています。このセミナーのきっかけに興味を持っていただけたらと思います。

「アジャイルな状態」はTDDよりも広い概念でアジャイルな状態になる方法のひとつとしてTDDがあると言えます。そういったもうちょっと引いた視点から個人だけでなくチームをうまく導く手法やその実例の講演などがありました。

オープンセミナーは来年も企画されており、実行委員長という立場は別の方がされる予定です。私は普通のスタッフとして来年も手伝いをしたいと思います。

講演の録画やスライドも随時アップされていくと思いますので、興味がある方は見てみてください。
